# Privacy Boundary

This repository is public. Treat it as a release artifact, not as a backup.

## Included

- Generic scripts
- Public-safe wrappers
- Documentation
- Redacted fixture data

## Excluded

- Live Home Assistant exports
- Private `.env` files
- Current project folders
- Generated outputs
- Backups
- Pulled dashboard data
- Any house-specific naming that is not clearly framed as an example

## Rule of thumb

If a file would reveal the current state of your house, devices, or repo
layout, do not publish it here.
